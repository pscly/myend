## 运行

1. 安装python
2. pip install -r requirements.txt
3. python app.py

## 功能

1. 通过接受请求，来将ip写入数据库，或者文件中，让我能够进行ip地址查看，或者是其他情况

       1. is_y: 必须, 而且是1
       2. who: 可选 我是谁
